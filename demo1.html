git status 查看工作区状态 git add + 文件名 将文件提交到暂存区 git add . 将全部修改过的文件存到暂存区 git commit 将暂存区的文件提交到版本库 git commit -m "注释" 将暂存区的文件提交到版本库，同时加上注释 git commit -a -m "注释" 将工作区的文件存到暂存区和版本库，并且同时加上注释 git log 查看提交记录 对比 git diff 工作区和暂存区的内容对比 git diff --cached 暂存区和版本库的内容对比 git diff master 工作区和版本库的内容对比 撤销 git reset HEAD 文件名 从暂存区撤回到工作区 git checkout -- 文件名 将工作区的内容撤回到暂存区或者版本库 git commit -m "注释" --amend 将本次提交和上一次提交到版本库的提交合并 删除 git rm --cached 文件名 在暂存区将对应的文件删掉 git rm -f 文件名 工作区和暂存区对应的文件一起删掉 git rm 文件名 在工作区将对应的文件删掉（也可直接删掉） 恢复 git checkout id 文件名 恢复某个对应的文件 git reset --hard id 恢复到对应的版本
